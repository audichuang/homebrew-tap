# Homebrew Tap

Custom Homebrew tap for [audichuang](https://github.com/audichuang)'s applications.

## Usage

```bash
brew tap audichuang/tap
brew install --cask skills-hub
```

## Available Casks

| Cask | Description |
|------|-------------|
| `skills-hub` | Manage and sync AI coding skills across tools |

## Updating

```bash
brew upgrade --cask skills-hub
```
